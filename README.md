# Weather Application 🌤️

A sleek, lightweight desktop weather application built using **Python** and **Tkinter**. It fetches real-time weather data from an external Weather API to provide current conditions for any city worldwide.

---

## ✨ Features

* **Real-Time Data:** Fetches up-to-date weather details using a live API.
* **Comprehensive Metrics:** Displays temperature, humidity, wind speed, and weather descriptions.
* **Search Functionality:** Quickly look up weather details by entering any city name.
* **User-Friendly GUI:** Clean and responsive desktop interface built with Tkinter.
